# retail-sales-analytics
#  Retail Sales Data Analytics Project  
**Tools Used:** Python, NumPy, Pandas, Matplotlib, Seaborn  
**Dataset Size:** 5,000 Rows  
**Domain:** Retail / Business Analytics  
**Author:** Charan Kumar  

---

##  Project Overview  
This project analyzes **5,000+ retail transactions** to uncover business insights related to:  
- Top-selling products  
- High-performing regions  
- Customer purchasing behavior  
- Seasonal sales patterns  
- Revenue drivers  

The goal is to help a retail business understand its performance and make data-driven decisions.

---

##  Dataset Description  
The dataset contains the following columns:

| Column Name     | Description |
|----------------|-------------|
| OrderID        | Unique order number |
| Date           | Order date |
| Product        | Product sold |
| Category       | Product category |
| Quantity       | Units sold |
| Price          | Price per unit |
| CustomerType   | New / Returning customer |
| Region         | Region of purchase |

* The dataset was generated synthetically for educational purposes.

##  Key Insights

Some examples (your actual output may vary):

- **Electronics** is the highest revenue-generating category.  
- **Returning customers** spend significantly more than new customers.  
- **Q4 months** showed the highest sales due to seasonal demand.  
- **South region** contributes the highest revenue.  
- Top products: *TV, Laptop, Mobile, Refrigerator*.  

---

##  Business Recommendations
- Increase inventory for top-selling products.  
- Provide targeted discounts for underperforming categories.  
- Allocate more marketing budget to high-revenue regions.  
- Launch loyalty programs to increase returning customers.  
- Boost stock during seasonal high-demand months.
